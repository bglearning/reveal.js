Pandas

Python Library for Data manipulation and analysis

  • Easy read and write from different sources/formats
  • Easy to do common tasks like handling missing data
  • Well-suited for tabular data with heterogeneously-typed columns
  • Plays well with Numpy and other libraries

Series

  • One-dimensional array-like structure
  • Capable of holding any data type
  • Has indices
  • Basically, a "smart" array

DataFrame

  • Two-dimensional tabular data structure
  • Capable of holding any data type
  • Has indices and columns

Indices are rows, Columns are... columns
